# Route Pages (`src/pages/`)

This directory contains the primary top-level route/page views for the E-Summit 2026 application.

## Files Added

- _None (All pre-existing pages were refactored for DRY abstractions)._

## Current Files / Refactored

- **`Home.jsx`**: E-Summit homepage showing countdown timers, schedule summaries, carousel grid events, sponsors, and FAQs.
- **`Buy.jsx`**: Passes booking selection grid screen.
- **`Events.jsx`**: Full listing grid view of all events.
- **`EventDetails.jsx`**: Detailed page layout showing schedules, format details, and descriptions for individual events.
- **`Schedule.jsx`**: Interactive schedule grid view outlining day-by-day itineraries.
- **`AdminAuth.jsx`**: Restricted access login flow.
- **`AdminDashboard.jsx`**: Admin management panels for tracking orders, marking passes sold-out, and scanning guest QR ticket entries.
- **`NotFound.jsx`**: The 404 fallback page when visitors navigate off the racing line.
