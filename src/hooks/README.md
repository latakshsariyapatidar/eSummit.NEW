# Custom React Hooks (`src/hooks/`)

This directory contains custom React hooks used across the E-Summit 2026 application.

## Files Added

- **`useDocumentTitle.js`**: Updates the browser document title dynamically.
- **`useLocalStorage.js`**: Syncs a React state variable directly with browser `localStorage`.
- **`useTransitionNavigate.js`**: Programmatically triggers the custom GSAP page transition before redirecting to a new route.

## Pre-existing Files

- **`useIsMobile.js`**: Responsive layout hook to detect mobile viewport width dynamically.
